Search Tools

You can search Congress.gov using the search bar available on most pages, a collection-specific search form, or advanced search. Advanced search gives you the option of using a form optimized for legislation, a query builder for customized searches, or a command line entry box for using SOLR query syntax. Additional help on searching is listed on the left.

Use fields to control where your search term or phrase is found in a document. Field searching is available from the search bar, the words & phrases box on search forms and the advanced search command line. Expand the accordion below for a complete list of fields, search operators, and examples.

 | 

To search, enter the prefix and number with or without periods and in upper or lowercase.

Bill, resolution and amendment numbers include an alphabetical prefix and a number. To search, enter the prefix and number with or without periods and in upper or lowercase. To eliminate hits on related bills or bill numbers in summaries, use the field label cite:. For example, cite:”hr933″.

Committee reports include an alphabetical prefix, Congress number and report number. Enter only the report number without a prefix when using the Report Numbers search box on the Committee Reports search form.

Public laws include an alphabetical prefix, Congress number and law number.

Search operators for use on all search forms except Legislation Text.

Search operators must be entered in uppercase.

Operator Use Examples
space between words (no operator) Defaults to AND operator. Finds results that contain all search terms. elementary secondary
” ” (quotation marks) Use quotation marks around two or more words to find words or word variants in an exact order. “polar bear”
OR Use between search terms to find results containing at least one of the terms. college OR university
AND Use between search terms (words or phrases) to find results containing both. elementary AND secondary
+ (plus sign) Use before a search term to find results containing that term. +student
NOT

– (minus sign)

 Use before a search term to find results that do not include the term. sanctuary NOT immigration

sanctuary -immigration
( )

(parenthesis)

 Use around search terms to group terms when using OR, AND and NOT. (disaster OR emergency) AND (funding OR appropriations)
~ (tilde) Use with a number after two or more words enclosed in quotation marks to find results containing all words (or word variants) within the specified distance from each other.

NOTE: Do not include ‘wildcard’ characters (? or *) within a proximity search query.

 “enhanced body armor”~10
? (question mark) Use in the middle or end of a word to indicate a single character ‘wildcard’. Finds results that include the search term with a single character variant.

NOTE: When using from search forms, be sure to uncheck Word Variants underneath the Words entry box.

 wom?n
* (asterisk) Use in the middle or end of a word to indicate a multiple character ‘wildcard’. Finds results that include the search term with zero or more character variants.

NOTE: When using from search forms, be sure to uncheck Word Variants underneath the Words entry box.

 approp*

Search operators for use on Legislation Text search form.

Operator Use Examples
space between words (no operator) Finds results that contain all search terms. elementary secondary
” ” (double quotation marks) Use double quotation marks around two or more words to find those words in an exact order. “investment incentives”
‘ ‘ (single quotation marks) Use single quotation marks for literal search. Use cases include searching for United States Code citations, percentages, and words and phrases where periods, commas, and hyphens are an important part of the query.

NOTE: Words at the end of a sentence are indexed with the trailing period. See “Wildcard within phrase” under “Operator Combinations” below for a method for finding phrases with or without a trailing period.

 ‘50% of the total amount appropriated’

’42 U.S.C. 651′

‘availability, access, and utilization’

‘audio-only’
~ (tilde) The OR operator. Use between search terms to find results containing at least one of the terms. ~(superstorm hurricane)
~(’42 U.S.C. 651′ ’42 U.S.C. 652′ ’42 U.S.C. 653′)
+ (plus sign) The MUST operator. Use before a search term to find results containing that term. +student

+rivers +lakes
! (exclamation point) The NOT operator. Use before a search term to find results that do not include the term. sanctuary !immigration
( )

(parenthesis)

 Use around search terms to group terms to apply an operator to all terms in the group. ~(disaster emergency)

+(rivers lakes)
N/ Use with a number before two or more words enclosed in parenthesis to find results containing all words within the specified distance from each other in any order. N/25(pesticide insect)
W/ Use with a number before two or more words enclosed in parenthesis to find results containing all words within the specified distance from each other in the exact order. W/3(secretary education)
* (asterisk) Use at the end of a word to indicate a multiple character wildcard. Finds results that include the search term with zero or more character variants. Wildcards are restricted to suffix wildcards. approp*
Operator Combinations Examples
Wildcard within proximity search w/10(private invest* corporation)
Proximity search with phrases n/20(“transmitted to congress” “annual report”)
Proximity search with combinations of phrases and keywords n/25(“95 percent” renewable)
Proximity search combined with SHOULD operator ~w/5(gun violence) ~w/5(firearm violence)
Wildcard within phrase w/1(windfall elimination provision*)
